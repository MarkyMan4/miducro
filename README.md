# Miducro

Play [here](https://markyman4.github.io/miducro/)

# Gameplay

- Use WASD to move
- Point and click with the mouse to shoot
- As you kill bugs, they may drop an item. Collect the item and you will see it show in the bottom left of the screen. Press E to use your item
